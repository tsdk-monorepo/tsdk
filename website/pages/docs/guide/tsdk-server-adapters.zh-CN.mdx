# tsdk-server-adapters

**tsdk** 目前提供 HTTP 和 WebSocket（Socket.IO） 两种模式的适配器：

- Express 适配器
- Fasitfy 适配器（其实复用 Express 适配器）
- Hono 适配器
- Socket.IO 适配器

## 自定义适配器

没有想要的适配器？

请参考 [**tsdk-server-adapters**](https://github.com/tsdk-monorepo/tsdk/tree/main/packages/tsdk-server-adapters) 代码，或者提交 issue 反馈。
