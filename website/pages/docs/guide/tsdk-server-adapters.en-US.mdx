# tsdk-server-adapters

**tsdk** provides HTTP and WebSocket（Socket.IO）protocol adapters：

- Express adapter
- Fasitfy (Reuse the express adapter)
- Hono adapter
- Socket.IO adapter

## Custom adapter

Please check [**tsdk-server-adapters**](https://github.com/tsdk-monorepo/tsdk/tree/main/packages/tsdk-server-adapters), or submit issue which adapter you need.
